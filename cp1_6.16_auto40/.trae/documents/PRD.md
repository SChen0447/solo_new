## 1. 产品概述

在线代码编辑器与沙箱运行平台，为在线教育场景提供安全的编程考试环境。考生可在浏览器中编写HTML/CSS/JavaScript代码并实时预览运行结果，后台可监控异常操作。平台通过沙箱技术确保代码执行安全，支持多语言模板切换，提供完善的控制台输出和错误定位功能。

- 核心价值：提供安全、高效、体验优秀的在线编程环境，支持教育考试场景下的代码编写、运行和评估
- 目标用户：在线教育平台的考生和考试管理人员

## 2. 核心功能

### 2.1 功能模块

1. **代码编辑模块**：基于CodeMirror的代码编辑器，支持语法高亮、自动缩进、括号匹配、主题切换
2. **沙箱运行模块**：基于iframe sandbox的安全执行环境，隔离用户代码与父页面
3. **状态管理模块**：管理代码模板、用户代码、运行结果和运行状态
4. **输出面板模块**：控制台日志展示，支持时间戳、错误高亮、一键清空
5. **模板选择模块**：多语言模板切换，自动填充预设代码

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 代码编辑器主页面 | 顶部工具栏 | 模板选择下拉菜单、运行按钮、主题切换按钮 |
| 代码编辑器主页面 | 左侧编辑区 | CodeMirror代码编辑器，支持语法高亮和实时输入 |
| 代码编辑器主页面 | 右侧预览区 | iframe沙箱运行结果预览，支持拖拽调整大小 |
| 代码编辑器主页面 | 底部输出面板 | 控制台日志输出，正常/错误分色显示，时间戳标记 |

## 3. 核心流程

用户选择代码模板 → 编辑器自动填充预设代码 → 用户编写/修改代码 → 点击运行按钮 → 沙箱执行代码 → 捕获控制台输出和运行时错误 → 更新预览区域和输出面板 → 用户查看结果或继续编辑

```mermaid
    G --> H["更新预览区"]

    I --> J["用户查看结果"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深色主题，背景#1e1e1e，编辑区#252526，边框#333
- **强调色**：绿色#28a745（运行按钮），红色#dc3545（错误提示），浅蓝色#4da6ff（拖拽分隔线）
- **按钮风格**：圆角4px，悬停平滑过渡0.2秒，运行时禁用状态带旋转加载动画
- **字体**：等宽字体 'Fira Code', 'Cascadia Code', monospace
- **布局风格**：左右分栏（桌面70%/30%），上下布局（移动端），可拖拽分隔线
- **动效风格**：所有交互元素0.2秒平滑过渡，主题切换0.3秒淡入淡出，加载时脉冲动画

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 主页面 | 顶部工具栏 | 深色背景#1e1e1e，左侧模板下拉选择器，中间运行按钮（绿色），右侧主题切换按钮 |
| 主页面 | 编辑区域 | 深灰背景#252526，行号区#2d2d2d，白色光标闪烁，语法高亮着色 |
| 主页面 | 预览区域 | 浅灰边框#555，内阴影效果，加载时中心蓝色脉冲动画 |
| 主页面 | 输出面板 | 深色背景，白色正常日志，红色错误日志，灰色时间戳HH:mm:ss |
| 主页面 | 分隔线 | 可拖拽，悬停浅蓝色高亮，拖拽时半透明效果 |

### 4.3 响应式设计
- **桌面端（>768px）**：左右布局，编辑区70%，预览区30%，可拖拽调整比例
- **移动端（≤768px）**：上下布局，编辑区在上，预览区在下，高度各50%
- **触控优化**：按钮最小高度44px，下拉菜单触控区域放大

### 4.4 性能指标
- 编辑器输入响应延迟：≤30ms（2000行代码内）
- 运行按钮到iframe预览加载：≤2秒（初始模板）
- 控制台日志条目超过200条时，自动截断最旧50条
