## 1. 产品概述

轻量级在线项目复盘报告自动生成器，解决团队在项目复盘和周报总结时手动排版费时、格式不统一的问题。用户以卡片形式录入工时、完成事项、阻塞问题等数据，一键生成带有图表和时间线、格式规范可导出的复盘报告。

- 目标用户：项目经理、团队负责人、开发/测试工程师等需要定期进行项目复盘的团队成员
- 核心价值：将零散的工时记录和事项信息自动整理成结构化报告，减少80%的手动排版时间

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 团队成员 | 直接使用 | 录入工时、添加事项、生成和导出报告 |

### 2.2 功能模块

1. **主看板页面**：左侧边栏导航（工时、事项、报告），右侧主内容区，顶部时间范围选择器和操作按钮
2. **工时录入模块**：卡片式工时记录录入，支持快速复制、多时段选择
3. **事项管理模块**：完成事项和阻塞问题两类卡片，支持拖拽排序和删除
4. **报告预览与导出模块**：图表展示（环形图、柱状图）、时间线、模板切换、一键导出HTML/PNG

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主看板 | 左侧边栏 | 深色背景，含功能图标导航（工时、事项、报告），移动端折叠为底部标签栏 |
| 主看板 | 顶部工具栏 | 固定栏，显示当前时间范围选择器（日/周/月/自定义）和"生成报告/导出"按钮 |
| 主看板 | 工时录入区 | 卡片形式添加工时记录，支持快速复制上一条的任务名称和类型 |
| 主看板 | 事项管理区 | 完成/阻塞两类卡片，拖拽排序、删除、优先级/状态设置 |
| 主看板 | 报告预览区 | 环形图（类型占比）、柱状图（每日分布）、时间线展示，模板切换 |
| 主看板 | 导出功能 | 一键生成带封面目录的HTML报告，支持PNG导出 |

## 3. 核心流程

用户打开应用 → 选择时间范围 → 录入工时记录（支持快速复制） → 添加完成事项（设置优先级、绩效标记） → 添加阻塞问题（设置状态） → 查看自动汇总的图表 → 选择报告模板 → 点击"生成报告" → 预览报告 → 导出HTML/PNG

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：冷静灰蓝 `#4A6FA5`，高亮色：暖黄 `#F5A623`
- 卡片背景：磨砂玻璃效果（半透明 + `backdrop-filter: blur(8px)`），边缘细微发光
- 按钮风格：圆角矩形，主按钮为灰蓝底色，次按钮为透明边框
- 字体：中文使用系统字体，英文使用 `DM Sans`，标题加粗
- 布局风格：仪表盘式布局，左侧窄边栏 + 右侧主内容区
- 图标风格：线性图标（Lucide）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主看板 | 左侧边栏 | 深色背景(#1a2332)，图标导航，hover发光效果，当前选中高亮 |
| 主看板 | 顶部工具栏 | 固定定位，时间范围选择器(日/周/月/自定义)，生成报告/导出按钮 |
| 主看板 | 工时卡片 | 磨砂玻璃效果，任务名称/工时/类型/描述，复制按钮，删除按钮 |
| 主看板 | 事项卡片 | 完成事项：优先级色标+绩效勾选框；阻塞问题：状态标签 |
| 主看板 | 图表区 | Recharts环形图+柱状图，入场动画（滑入渐显），切换滑动效果 |
| 主看板 | 报告预览 | 简洁商务风/创意风格两套模板，封面+目录+图表+时间线 |

### 4.3 响应式设计

- 桌面端（≥768px）：左侧边栏 + 右侧主内容区，图表双列布局
- 移动端（<768px）：左侧边栏折叠为底部标签栏，图表单列布局，卡片间距缩小
- 卡片列表支持纵向弹性滚动（`overflow-y: auto`），头部固定

### 4.4 动画设计

- 图表入场：从底部滑入并渐显
- 图表切换：旧图表向右滑出、新图表向左滑入
- 卡片操作：拖拽时轻微缩放+阴影增强
- 页面切换：淡入淡出过渡
- 环形图：旋转进入动画
