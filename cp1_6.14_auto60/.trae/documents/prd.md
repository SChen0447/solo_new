## 1. 产品概述

私房菜经营管理助手——面向个人厨师与私房菜经营者的全流程管理工具，解决客户需求分散导致菜单调整频繁记错遗漏、采购食材过量浪费两大核心痛点。通过统一的数字化平台实现客户预约、菜单定制、食材库存三位一体管理。

- 目标用户：个人厨师、私房菜工作室经营者
- 核心价值：减少菜单错漏、降低食材浪费、提升客户满意度

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 厨师/经营者 | 直接使用 | 管理客户、预约、菜单、食材库存全部功能 |

### 2.2 功能模块

1. **预约管理页**：客户信息录入、预约创建、纵向时间线视图、状态标记
2. **菜单管理页**：自动菜品推荐、4道菜菜单生成、卡片展示、菜品替换
3. **库存管理页**：食材录入、库存预警、分类筛选、关键字搜索

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 预约管理 | 客户信息卡 | 记录客户姓名、联系方式、饮食偏好（素食/无麸质/低卡/无限制） |
| 预约管理 | 预约创建表单 | 填写用餐日期、人数、预算范围（每客100-500元） |
| 预约管理 | 时间线视图 | 纵向布局每30分钟刻度，当天预约高亮，过期未确认灰色标记+已过天数 |
| 菜单管理 | 菜单生成 | 根据偏好+预算自动推荐4道菜组合（前菜+汤品+主食+甜品） |
| 菜单管理 | 菜单卡片 | 菜品类型标签（绿前菜/蓝汤品/橙主食/紫甜品），超预算20%红色边框 |
| 菜单管理 | 菜品替换 | 手动替换菜品，自动从库存寻找替代选项 |
| 库存管理 | 食材列表 | 名称、类别（蔬菜/肉类/海鲜/乳制品/干货/调味品）、库存量、预警线 |
| 库存管理 | 预警标识 | 低于预警线食材行首红色闪烁圆点 |
| 库存管理 | 筛选搜索 | 按类别筛选、关键字搜索 |

## 3. 核心流程

**预约→菜单→库存主流程**：
1. 厨师录入客户信息和饮食偏好
2. 为客户创建预约，填写日期、人数、预算
3. 系统根据偏好和预算从食材库存自动推荐菜单
4. 厨师可手动替换某道菜，系统自动寻找替代选项
5. 确认菜单后，相关食材从库存中扣减
6. 库存低于预警线时自动提醒补货

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：温暖米白（#FAF6F1）+ 橄榄绿（#6B8F71）
- 辅助色：深棕（#5C4033）、浅卡其（#E8DFD0）
- 强调色：暖橙（#D4915E）用于CTA按钮和关键操作
- 按钮风格：圆角（8px），轻微阴影，hover上浮效果
- 字体：Noto Serif SC（标题）+ Noto Sans SC（正文）
- 布局风格：左侧导航栏 + 右侧内容区，卡片式布局
- 图标风格：线性图标（lucide-react），与橄榄绿主色调统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 预约管理 | 时间线视图 | 纵向时间线，30分钟刻度，当天高亮绿色，过期灰色虚线，未确认标记天数 |
| 预约管理 | 客户卡片 | 圆角卡片，hover上浮阴影，饮食偏好标签彩色显示 |
| 预约管理 | 创建表单 | 侧边抽屉滑出，表单字段含验证 |
| 菜单管理 | 菜单卡片 | 4张卡片并排，左上角类型标签色块，淡入淡出0.3s，逐张展开0.15s间隔 |
| 菜单管理 | 替换弹窗 | 居中模态框，展示可替换菜品列表 |
| 库存管理 | 食材列表 | 表格布局，低库存行红色闪烁圆点，分类Tab筛选 |
| 库存管理 | 搜索栏 | 顶部固定搜索框+类别下拉 |

### 4.3 响应式设计

- 桌面优先（≥768px）：侧边导航 + 内容区双栏布局
- 移动端（<768px）：单列布局，侧边导航折叠为汉堡菜单
- 触摸优化：按钮最小44px点击区域，卡片间距增大

### 4.4 交互动画

- 卡片点击：轻微上浮阴影效果（box-shadow过渡0.2s）
- 菜单卡片切换：淡入淡出过渡0.3s
- 菜单生成：卡片逐张展开（每张间隔0.15s）
- 预约时间线：流畅滚动（帧率≥50fps）
- 库存预警圆点：CSS闪烁动画
