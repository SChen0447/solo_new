## 1. 产品概述

指尖琴音是一款交互式钢琴教学应用，让用户通过电脑键盘或鼠标点击虚拟钢琴键盘来学习和练习简单乐曲。应用提供乐谱跟随演奏、演奏录制与回放、多套视觉主题等功能，营造沉浸式的在线音乐学习体验。

- 目标用户：音乐初学者、钢琴爱好者、希望用趣味方式学习简单乐曲的用户
- 核心价值：零门槛上手、即时音频反馈、乐谱引导式学习

## 2. 核心功能

### 2.1 功能模块

1. **主页面**：虚拟钢琴键盘、乐谱显示区、左右控制面板

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 钢琴键盘 | 渲染C3-B6共4个八度的52键键盘，支持鼠标点击和电脑键盘映射，按键时播放对应音高并显示动画 |
| 主页面 | 乐谱显示 | Canvas渲染滚动乐谱，高亮当前演奏音符，支持3首内置曲目（小星星、欢乐颂、两只老虎） |
| 主页面 | 左侧控制面板 | 歌曲选择下拉框、开始学习/停止学习按钮 |
| 主页面 | 右侧控制面板 | 录音/回放按钮、回放速度调节（0.5x/1x/2x） |
| 主页面 | 主题切换 | 3套预设配色主题切换，CSS过渡动画0.4秒，存储到localStorage |

## 3. 核心流程

### 3.1 自由演奏流程
用户点击琴键或按下键盘 → 触发Web Audio API播放音高 → 琴键高亮动画 → 音符状态更新

### 3.2 乐谱学习流程
用户选择歌曲 → 点击"开始学习" → 乐谱自动滚动并高亮当前音符 → 用户按正确顺序弹奏 → 绿色闪烁确认/红色闪烁报错 → 继续下一音符

### 3.3 录制回放流程
用户点击"录音" → 记录按键序列（音符+时间戳） → 最多60秒 → 点击"回放" → 自动演奏并同步高亮 → 支持速度调节

## 4. 界面设计

### 4.1 设计风格

- 主背景色：深蓝灰色 `#1a1a2e`
- 主色调：青蓝色到紫色渐变
- 按钮样式：圆角矩形，渐变背景，悬停亮度+10%放大1.05倍，点击缩小0.95倍
- 字体：系统无衬线字体（Inter或系统默认）
- 动画：缓入缓出CSS过渡，时长0.15-0.4秒

### 4.2 页面设计总览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主页面 | 钢琴键盘 | 白键象牙白渐变、黑键深灰渐变、按下动画(95%缩小+深阴影)、悬浮变色 |
| 主页面 | 乐谱区 | Canvas滚动乐谱、当前音符高亮、半透明分隔线 |
| 主页面 | 左面板 | 歌曲选择器、开始/停止学习按钮、半透明背景 |
| 主页面 | 右面板 | 录音/回放按钮、速度调节、半透明背景 |
| 主页面 | 主题切换 | 3色主题按钮组 |

### 4.3 响应式适配

- 桌面优先设计
- 宽度<768px：键盘缩至70%、控制面板移至键盘下方、按钮横向排列、乐谱竖排
- 触摸优化：琴键点击区域适配
