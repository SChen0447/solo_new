## 1. 产品概述

食谱探险家是一款互动式美食发现应用，让用户像翻阅美食杂志一样探索全球美食。通过随机推荐、标签筛选、个人收藏和分享功能，为用户提供沉浸式的烹饪灵感。

- **核心价值**：打破选择困难症，通过随机推荐和智能标签筛选，帮助用户快速发现适合的食谱
- **目标用户**：家庭主妇、烹饪爱好者、年轻上班族
- **市场定位**：轻量级美食发现工具，注重交互体验和视觉美感

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览食谱、标签筛选、收藏管理、生成分享链接 |

### 2.2 功能模块

1. **首页**：随机推荐食谱、标签筛选栏、食谱卡片网格
2. **收藏夹页面**：收藏食谱列表、拖拽排序、分享功能
3. **食谱详情模态框**：完整食谱信息展示、步骤说明

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 随机推荐区 | 每次刷新随机展示3道食谱卡片，带淡入上滑动画 |
| 首页 | 标签筛选栏 | 8个美食标签（快捷、甜品、辣味、素食、早餐、烧烤、汤品、海鲜），点击后平滑切换食谱列表 |
| 首页 | 食谱卡片网格 | 展示食谱图片、名称、时长、难度，带悬停上浮效果 |
| 收藏夹页面 | 收藏网格 | 每行3张卡片，支持拖拽调整顺序 |
| 收藏夹页面 | 分享功能 | 生成24小时有效的分享短码链接 |
| 模态框 | 食谱详情 | 展示完整食谱信息，含步骤列表，带缩放进入动画 |

## 3. 核心流程

用户打开首页 → 看到随机推荐的3道食谱 → 点击标签筛选感兴趣的分类 → 点击食谱卡片查看详情 → 点击心形按钮收藏 → 进入收藏夹页面管理收藏 → 拖拽调整顺序 → 生成分享链接分享给好友

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白色 #F5F0E8（温暖舒适的背景）
- **辅色调**：橙色 #E67E22（食欲感、活力）
- **字体色**：深灰色 #333（清晰易读）
- **卡片渐变**：浅橙到浅黄（温暖美食氛围）
- **按钮风格**：圆角按钮，点击有按压缩放反馈（0.95倍，0.1秒）
- **字体选择**：显示字体使用 Playfair Display（优雅杂志感），正文字体使用 Lora（舒适阅读）
- **布局风格**：卡片式布局，最大宽度1200px居中，留白充足
- **图标风格**：使用 lucide-react 线性图标，简洁现代

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 首页 | 顶部导航 | Logo、导航链接（首页/收藏夹）、暖色调背景 |
| 首页 | 标题区域 | "今日美食推荐"大标题，优雅字体，橙色装饰线 |
| 首页 | 标签筛选栏 | 圆角标签按钮，选中状态橙色填充，水平滚动 |
| 首页 | 食谱卡片 | 渐变背景，圆角，悬停上浮阴影，底部上滑淡入动画 |
| 收藏夹页面 | 收藏列表 | 网格布局，拖拽时半透明，平滑重排动画 |
| 收藏夹页面 | 分享按钮 | 橙色主按钮，生成短码后显示复制提示 |
| 模态框 | 详情弹窗 | 背景模糊，中心缩放进入，优雅的步骤列表 |

### 4.3 响应式设计

- **桌面端**（>768px）：卡片每行3张，标签横向排列
- **移动端**（<768px）：卡片每行1张，标签行水平滚动，导航简化
- **触摸优化**：按钮最小高度44px，拖拽手势流畅

### 4.4 动画与交互

- **页面加载**：卡片从底部上滑淡入（0.4秒），交错延迟
- **标签切换**：旧卡片向左淡出，新卡片从右侧滑入，每张延迟0.1秒
- **收藏按钮**：心跳缩放动画（0.5秒），红色填充
- **卡片悬停**：上浮5px，阴影增强（0.3秒过渡）
- **模态框**：从0.8倍缩放到1倍（0.3秒），背景模糊
- **拖拽排序**：拖拽时半透明，其他卡片自动平滑重排
