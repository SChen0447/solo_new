## 1. 产品概述

虚拟宠物试养评估应用——帮助用户在领养宠物前，通过填写生活问卷生成虚拟宠物，模拟其日常行为并推送护理任务，辅助用户做出科学的领养决策。

- 目标用户：有意向领养猫/狗但不确定是否合适的潜在领养人
- 核心价值：降低冲动领养率，通过沉浸式虚拟试养体验让用户提前感受养宠日常

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 填写问卷、虚拟试养、查看历史、导出报告 |

### 2.2 功能模块

1. **问卷页面**：选择宠物种类、填写生活作息与居住条件问卷
2. **模拟页面**：宠物状态实时模拟、护理任务推送、日历视图、报告导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 问卷页面 | 宠物种类选择 | 猫/狗单选切换，动态展示对应问卷题目 |
| 问卷页面 | 生活问卷表单 | 居住空间大小（滑块）、平均离家时长（滑块）、是否有其他宠物（单选）、运动频率（单选）、作息时间（滑块）、养宠经验（单选），至少5题 |
| 问卷页面 | 表单验证 | 必填项未填时红色边框高亮+0.3秒抖动动画 |
| 模拟页面 | 宠物头像展示 | 猫为圆形、狗为椭圆形卡通头像，眼睛大小与活跃度关联 |
| 模拟页面 | 状态条 | 四条垂直进度条（饱腹度/精力/社交/卫生），高度200px宽度20px，颜色渐变#4CAF50→#FF5722，每5分钟自动增减 |
| 模拟页面 | 任务卡片 | 4-6个护理任务，包含图标/名称/倒计时(mm:ss)，未完成#FFF3E0，完成#E8F5E9+绿色勾选动画 |
| 模拟页面 | 操作确认弹窗 | 半透明遮罩#00000080，弹窗居中360px宽，圆角12px，关闭时缩小消失0.3秒 |
| 模拟页面 | 周日历视图 | 每天格子显示完成任务数/总任务数，已完成#81C784，未完成#FFCDD2，点击展开详情列表 |
| 模拟页面 | 速度控制 | 1x/2x/4x按钮，加速时间流速（状态动画仍平滑过渡） |
| 模拟页面 | 导出报告 | 问卷摘要+7天状态折线图+任务完成率饼图，PDF格式下载 |

## 3. 核心流程

用户打开应用 → 选择宠物种类（猫/狗）→ 填写生活问卷（5+题）→ 提交验证 → 跳转模拟页面 → 系统生成虚拟宠物 → 宠物状态实时变化（饱腹度/精力/社交/卫生）→ 每日推送4-6个护理任务 → 用户点击任务卡片完成 → 任务状态更新 → 日历视图记录历史 → 可随时导出评估报告

## 4. 用户界面设计

### 4.1 设计风格

- 主色：暖灰色#37474F（导航栏）、柔和背景#F5F5F5
- 辅助色：进度条渐变#4CAF50→#FF5722、任务卡片#FFF3E0/#E8F5E9
- 按钮风格：Material Design圆角8px，hover缩放1.05（0.2秒），点击缩放0.98（0.1秒）
- 字体：系统默认sans-serif
- 布局：顶部固定导航栏60px + 左右双栏（40%/60%，间距32px）
- 圆角统一8px，卡片阴影0 2px 8px rgba(0,0,0,0.1)

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 问卷页面 | 宠物选择 | 两个大卡片（猫/狗），选中态带边框高亮和缩放 |
| 问卷页面 | 问卷表单 | 单选框组、滑块（带数值标签）、提交按钮#1976D2 |
| 模拟页面 | 导航栏 | 固定顶部60px，背景#37474F，白色文字，速度控制按钮组 |
| 模拟页面 | 宠物区（左40%）| 卡通头像+四个垂直进度条 |
| 模拟页面 | 任务区（右60%）| 任务卡片网格，倒计时，确认弹窗 |
| 模拟页面 | 日历区 | 周日历网格，展开式详情列表 |
| 模拟页面 | 报告按钮 | 右上角#1976D2背景白色文字按钮 |

### 4.3 响应式适配

- 桌面优先设计，屏幕宽度<768px时左右栏堆叠为上下排布
- 卡片宽度自适应100%，导航栏菜单折叠为汉堡图标
- 状态条和任务卡片在小屏幕上保持可读性

### 4.4 动效规范

- 表单验证抖动：0.3秒
- 任务完成勾选：0.4秒缩放+旋转
- 弹窗关闭：0.3秒从中心缩小消失
- 状态条渐变：0.5秒宽度过渡
- 按钮hover：0.2秒缩放1.05
- 按钮点击：0.1秒缩放0.98
