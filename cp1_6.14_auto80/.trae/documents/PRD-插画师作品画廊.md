## 1. 产品概述

插画师作品画廊与客户反馈收集应用，帮助自由插画师集中展示分散在各社交平台的作品，并收集客户反馈。
- 解决插画师作品分散、难以批量展示风格和获取客户反馈的痛点
- 目标用户：自由插画师（内容管理者）和潜在客户（浏览者、评论者）
- 产品价值：建立专业个人品牌形象，高效展示作品集，收集灵感启发和客户评价

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 插画师（管理员） | 无需登录（本地应用） | 添加/编辑/删除作品、切换主题、管理作品集 |
| 访客（客户） | 无需登录 | 浏览画廊、查看作品详情、提交留言反馈 |

### 2.2 功能模块

1. **作品管理模块**：作品添加表单、作品列表、编辑/删除操作
2. **画廊展示模块**：瀑布流作品网格、作品详情页、Swiper轮播图
3. **客户反馈模块**：反馈表单、评论列表、时间显示
4. **主题定制模块**：主题切换、全局样式应用、localStorage持久化
5. **导航模块**：侧边导航栏、移动端底部导航、路由管理

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 画廊首页 | 瀑布流展示 | 自适应3-4列瀑布流、作品卡片悬停效果、分页/排序筛选 |
| 作品详情页 | 轮播展示 | Swiper多图轮播、完整描述、风格标签、反馈区域 |
| 作品管理页 | 表单管理 | 添加作品表单（标题、描述、标签、图片URL）、作品列表、编辑删除 |
| 主题设置页 | 主题定制 | 4种主题选择（柔和暖色调/冷色调/复古色/高对比暗色）、实时预览 |

## 3. 核心流程

### 3.1 插画师添加作品流程
插画师进入作品管理页 → 填写作品表单（标题、描述、选择风格标签、添加图片URL）→ 提交保存 → 数据存入后端JSON → 画廊首页实时展示新作品

### 3.2 客户浏览与反馈流程
访客进入画廊首页 → 浏览瀑布流作品卡片 → 点击感兴趣的作品 → 进入详情页查看轮播图和完整描述 → 填写昵称和评论 → 提交反馈 → 评论立即显示在列表中

## 4. 用户界面设计

### 4.1 设计风格

**设计基调：艺术画廊风格，简约优雅，突出作品本身**

- **主色调**：根据主题动态变化
  - 柔和暖色调：米白背景 #FAF6F0，暖棕主题色 #B8860B
  - 冷色调：浅灰蓝背景 #F0F4F8，深蓝主题色 #2E4A62
  - 复古色：米黄背景 #F5EFE0，砖红主题色 #8B4513
  - 高对比暗色：深灰背景 #1A1A1A，金色主题色 #D4AF37
- **按钮风格**：圆角8px，填充式按钮，悬停有微妙缩放效果
- **字体**：标题使用 "Playfair Display" 衬线字体，正文使用 "Noto Sans SC" 无衬线字体
- **布局**：左右分栏（桌面），上下堆叠（移动端）
- **卡片风格**：圆角12px，柔和box-shadow (0 4px 20px rgba(0,0,0,0.08))，白色背景

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 画廊首页 | 瀑布流网格 | CSS columns布局、作品卡片、淡入错开动画、悬停遮罩效果 |
| 作品详情页 | 轮播展示 | Swiper轮播（fade过渡）、标签徽章、评论列表、表单验证 |
| 作品管理页 | 表单管理 | 多标签选择器、图片URL列表、确认删除对话框 |
| 主题设置页 | 主题选择 | 主题预览卡片、单选切换、实时应用效果 |

### 4.3 响应式设计

- **桌面端（>1024px）**：左侧240px固定导航栏，右侧内容区，瀑布流3-4列
- **平板端（768-1024px）**：导航栏折叠为底部60px高度，瀑布流2列
- **手机端（<768px）**：底部导航，瀑布流2列，轮播图占满屏幕宽度
- **触摸优化**：所有可点击元素最小44x44px，滑动手势支持轮播切换

### 4.4 动效设计

- **卡片加载**：opacity 0→1，持续0.3s，animation-delay错开300ms逐个出现
- **卡片悬停**：向上浮动2px，shadow加深，半透明深色遮罩淡入显示描述
- **轮播切换**：fade过渡效果，持续300ms
- **主题切换**：所有颜色属性transition 0.3s平滑过渡
- **表单聚焦**：边框颜色过渡为主题色，持续0.2s
