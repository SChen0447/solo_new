## 1. 产品概述

FitClass 是一款面向连锁健身房的私教课程预约与教练管理看板应用，解决会员预约私教课、教练管理课程与记录反馈、管理员统计运营数据的核心需求。
- 目标用户：健身房会员、私教教练、健身房管理员
- 核心价值：提升预约效率、优化教练排课管理、数据驱动运营决策

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 会员 | 选择已有会员账号登录 | 浏览教练、预约课程、查看历史记录 |
| 教练 | 选择已有教练账号登录 | 查看日程、标记出勤、添加课后评价 |
| 管理员 | 管理员账号登录 | 查看统计看板、预约数据、活跃度分析 |

### 2.2 功能模块

1. **预约页面**：教练列表与档案、时间段选择、课程预约、个人课程表
2. **教练日程页面**：未来7天课程列表、出勤标记、评价弹窗
3. **统计看板页面**：今日预约总数、本周活跃教练/会员数、7天预约趋势柱状图
4. **历史记录页面**：过去3个月预约历史、分页列表、状态标签

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 预约页面 | 教练卡片列表 | 展示教练头像、姓名、专长标签、星级评分，点击展开可预约时段 |
| 预约页面 | 时间段选择器 | 每天8:00-20:00整点时段，已约灰色不可选，可选蓝色，选30/45/60分钟时长 |
| 预约页面 | 预约确认 | 确认后生成预约ID，显示在个人课程表中 |
| 教练日程页面 | 课程列表 | 未来7天课程，显示会员头像、姓名、时间段 |
| 教练日程页面 | 出勤标记 | 点击出勤按钮，灰色→绿色+勾选图标 |
| 教练日程页面 | 评价弹窗 | 半透明遮罩+白色圆角弹窗，输入评价(150字)和会员状态(进步/保持/退步) |
| 统计看板页面 | 今日数据卡片 | 今日预约总数、本周活跃教练和会员数量 |
| 统计看板页面 | 趋势柱状图 | 过去7天每天预约人数，颜色从浅蓝渐变到深蓝，柱顶显示数字 |
| 历史记录页面 | 预约历史列表 | 过去3个月记录，分页(10条/页)，含教练昵称、日期、时长、状态标签 |

## 3. 核心流程

### 3.1 会员预约流程
会员登录 → 浏览教练列表 → 点击教练展开时段 → 选择空闲时段 → 选择课程时长 → 确认预约 → 系统生成预约ID → 时段变为已预约状态 → 预约显示在个人课程表

### 3.2 教练管理流程
教练登录 → 查看未来7天课程列表 → 点击出勤按钮 → 弹出评价弹窗 → 输入评价和会员状态 → 保存 → 更新课程状态为已出勤 → 同步更新预约记录

### 3.3 取消预约流程
会员查看课程表 → 选择未开课的预约 → 点击取消(开课前2小时) → 系统释放时段 → 时段恢复可选状态

## 4. 用户界面设计

### 4.1 设计风格
- 主色：活力蓝 #1976D2，辅色：活力橙 #FF9800
- 背景：浅灰 #F5F5F5，卡片/白底区域：白色
- 字体：系统无衬线字体，使用 rem 单位
- 布局：顶部固定导航栏(56px蓝色) + 左侧侧边栏(240px白色) + 主内容区
- 卡片阴影：0 2px 8px rgba(0,0,0,0.1)，悬停 0 4px 16px rgba(0,0,0,0.2) + 上移2px
- 按钮涟漪动画(0.4秒)，加载旋转圆环，错误toast右上角滑入3秒消失
- 运动风格简洁现代，强调活力和专业感

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局布局 | 顶部导航栏 | 蓝色背景#1976D2，白色文字，左侧Logo"FitClass"，右侧用户头像+名字 |
| 全局布局 | 左侧侧边栏 | 白色背景240px宽，菜单项：首页/预约/历史/统计，悬停蓝色背景白色文字，分割线列表 |
| 预约页面 | 教练卡片 | 白色卡片+阴影，头像+姓名+专长标签+星级评分，展开显示时段网格 |
| 预约页面 | 时段网格 | 灰色=已约，蓝色=可预约，每整点一个slot |
| 教练日程页面 | 课程列表项 | 白色卡片，会员头像+姓名+时间段，出勤按钮(灰→绿) |
| 教练日程页面 | 评价弹窗 | 深色半透明遮罩，白色圆角16px弹窗，文本输入+单选按钮 |
| 统计看板页面 | 数据卡片 | 白色卡片+图标+数字标签 |
| 统计看板页面 | 柱状图 | CSS柱状图，#BBDEFB→#1565C0渐变，柱顶数字 |
| 历史记录页面 | 列表项 | 白色行卡片，状态标签(绿=已出勤/红=未出勤/橙=已取消) |

### 4.3 响应式设计
- 桌面优先设计，手机端(宽度<768px)侧边栏收起到汉堡菜单
- 卡片在手机端变为全宽
- 导航栏在手机端简化
- 字体大小使用rem单位确保可访问性

### 4.4 数据一致性规则
- 教练标记出勤时，自动更新对应预约记录status为'attended'
- 同一slot不能重复预约
- 会员取消预约（开课前2小时），slot自动释放变为可选
