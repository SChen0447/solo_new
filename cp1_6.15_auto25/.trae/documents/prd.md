## 1. 产品概述
魔法药水酿造与效果演示应用——一个基于实时物理模拟的交互式魔法炼金工坊。用户在锅炉容器中拖拽添加魔法材料，系统实时模拟液体混合的物理行为（颜色扩散、密度分层、气泡沸腾），通过搅拌加速反应，配方正确时药水发出脉冲光效并变色，配方错误则产生烟雾或爆裂效果。

- 目标用户：奇幻/魔法爱好者、互动体验用户
- 核心价值：沉浸式魔法药水酿造体验，融合物理模拟与视觉特效

## 2. 核心功能

### 2.1 功能模块
1. **酿造工坊页面**：锅炉容器Canvas渲染、材料拖拽添加、木棍搅拌交互、配方匹配反馈、搅拌/温度计量表、材料百科面板

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 酿造工坊 | 锅炉容器 | Canvas 2D渲染锅炉（400×400px），粒子系统实时绘制液体，支持材料拖入和搅拌交互 |
| 酿造工坊 | 材料选择面板 | 底部水平滚动6种材料卡片，点击拖拽到锅炉 |
| 酿造工坊 | 搅拌/温度计量表 | 锅炉左侧两个垂直进度条（搅拌量、温度） |
| 酿造工坊 | 药水效果叠加 | 成功脉冲光晕+颜色渐变；失败烟雾+屏幕震动 |
| 酿造工坊 | 材料百科面板 | 锅炉下方可折叠面板，展示已解锁配方记录 |

## 3. 核心流程

用户从材料面板选择材料 → 拖拽到锅炉 → 材料粒子团生成并扩散 → 用户搅拌加速混合 → 系统自动检测配方 → 匹配成功：脉冲光效+变色；匹配失败：烟雾+震动 → 配方记录自动更新

```mermaid
flowchart TD
    "A[选择材料]" --> "B[拖拽到锅炉]"
    "B" --> "C[粒子团生成扩散]"
    "C" --> "D[搅拌加速混合]"
    "D" --> "E{配方检测}"
    "E" --> "F[成功：脉冲光效+变色]"
    "E" --> "G[失败：烟雾+震动]"
    "F" --> "H[配方记录更新]"
    "G" --> "I[继续添加/搅拌]"
    "I" --> "D"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深色魔幻风格（背景#1a1a2e，容器#16213e）
- 锅炉渐变：顶部深紫#0f3460到底部深蓝#1a1a2e
- 按钮样式：圆角半透明玻璃质感
- 字体：无衬线字体，最小12px
- 布局：中央锅炉+底部材料面板+左侧计量表+下方百科面板
- 动画：悬停0.2s过渡、脉冲光环8s旋转、闪烁红色边框

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 酿造工坊 | 锅炉容器 | 400×400px Canvas，CSS弧形锅壁锅耳，底部径向渐变光晕 |
| 酿造工坊 | 材料卡片 | 80×100px，圆角12px，半透明玻璃背景，悬停上浮6px |
| 酿造工坊 | 搅拌进度条 | 20×200px，底部深棕到顶部金色渐变 |
| 酿造工坊 | 温度条 | 20×200px，底部蓝色到顶部红色渐变 |
| 酿造工坊 | 百科面板 | 可折叠，展开250px高，网格列表 |

### 4.3 响应式适配
- 桌面优先，宽度<768px时锅炉缩至70%，材料面板改为垂直两行排列
- 所有交互元素支持鼠标和触摸操作
