## 1. 产品概述

三维虚拟化学实验室是一个面向化学爱好者的网页端 3D 交互式化学实验模拟平台。用户可以在逼真的 3D 实验台上拖拽试剂瓶倒入烧杯，观察化学反应的颜色变化、气泡产生和温度/pH 值变化，所有操作自动记录并支持回放。

- 目标用户：化学爱好者、中学化学教师和学生
- 核心价值：安全、直观、零成本地体验化学实验过程

## 2. 核心功能

### 2.1 功能模块

1. **实验台模块**：3D 场景，包含木质实验台面、烧杯、量筒、试管架，支持点击拖拽操作
2. **反应模拟模块**：根据试剂组合执行化学反应，展示颜色渐变、气泡、温度变化和 pH 值变化
3. **实验记录模块**：自动记录每步操作，支持点击回放和高亮对应区域

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 实验室主页面 | 实验台场景 | 3D 渲染实验台面(80x200单位，木质纹理)、烧杯(透明玻璃，内径10单位，容量200ml)、量筒、试管架，用户拖拽试剂瓶倒液 |
| 实验室主页面 | 试剂瓶与倒液 | 试剂瓶带标签和液面，拖拽时生成粒子流(30粒子，2单位/秒)，液面平滑上升 |
| 实验室主页面 | 反应模拟 | 预设5种化学反应(HCl+NaOH、FeCl₃+KSCN等)，颜色0.3秒缓动过渡，显示实时温度和pH值进度条 |
| 实验室主页面 | 实验日志 | 左侧日志列表，新条目0.3秒滑入动画，背景#F5F5F0，点击回放高亮操作区域 |

## 3. 核心流程

用户打开页面 → 查看 3D 实验台场景 → 从试剂架拖拽试剂瓶到烧杯上方 → 释放触发倒液动画(粒子流+液面上升) → 反应模拟引擎判断试剂组合 → 执行化学反应(颜色渐变+气泡+温度变化) → 自动记录操作日志 → 用户可点击日志条目回放操作

## 4. 用户界面设计

### 4.1 设计风格

- 主背景色：浅灰蓝 #E8ECF1
- 工具栏：暖白 #FFF8F0
- 半透明面板：玻璃质感 #CCD8E8
- 日志条目背景：#F5F5F0
- 试剂瓶标签字体：衬线字体 Georgia
- 面板风格：半透明玻璃拟态
- 布局：中央3D场景 + 左侧实验日志面板 + 右上角参数调节面板

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 实验室主页面 | 3D场景区 | Three.js画布，占主区域，木质台面，透明玻璃烧杯，试剂瓶 |
| 实验室主页面 | 实验日志面板 | 左侧半透明面板(#CCD8E8)，条目0.3秒滑入，Georgia标签 |
| 实验室主页面 | 实时数据面板 | 温度数值显示，pH值进度条 |
| 实验室主页面 | Lil-gui面板 | 右上角参数调节(加热温度、反应速率等) |

### 4.3 响应式设计

- 桌面优先设计
- 800px 宽度以下：3D场景缩小至屏幕高度80%，实验日志自动折叠为左侧隐藏抽屉
- 触摸操作支持拖拽倒液

### 4.4 3D场景指引

- **环境**：室内实验室氛围，暖色环境光
- **灯光**：环境光(0.6强度) + 方向光(0.8强度) + 点光源模拟台灯
- **相机**：透视相机，初始位置(0, 40, 60)，俯视30度角看向台面中心
- **材质**：台面使用木质纹理(procedural)，烧杯使用透明玻璃材质(MeshPhysicalMaterial, transmission, roughness=0.05)，液体使用半透明材质
- **交互**：OrbitControls旋转查看，DragControls拖拽试剂瓶
- **后处理**：抗锯齿，环境反射
- **性能预算**：60FPS，最多10个粒子系统，3个烧杯实时液体渲染

## 5. 预设化学反应

| 序号 | 反应方程式 | 现象描述 |
|------|-----------|----------|
| 1 | HCl + NaOH → NaCl + H₂O | 温度升高，烧杯底部冒泡，颜色不变 |
| 2 | FeCl₃ + KSCN → [Fe(SCN)]²⁺ | 0.5秒内渐变为血红色 |
| 3 | CuSO₄ + NaOH → Cu(OH)₂↓ | 蓝色沉淀生成 |
| 4 | Na₂CO₃ + HCl → NaCl + H₂O + CO₂↑ | 大量气泡产生 |
| 5 | AgNO₃ + NaCl → AgCl↓ | 白色沉淀生成 |
