## 1. 产品概述

在线食谱分享与推荐系统——让美食爱好者上传自己做的菜式图片和步骤，系统根据口味标签（酸甜、麻辣、清淡等）自动推荐相似菜谱，打造温暖的美食分享社区。

- 目标用户：热爱烹饪、希望发现新菜式的家庭厨师和美食爱好者
- 核心价值：通过口味标签智能推荐，让用户轻松发现相似风味的菜谱，降低"今天吃什么"的决策成本

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 用户名+密码注册 | 浏览菜谱、发布菜谱、收藏、打分、搜索 |
| 未登录访客 | 无 | 浏览菜谱、搜索 |

### 2.2 功能模块

1. **首页**：公共食谱墙、搜索框、口味标签筛选
2. **食谱详情页**：分步展示、收藏、打分（1-5星）、相似推荐
3. **发布页**：新菜谱发布表单
4. **我的收藏页**：收藏列表管理
5. **登录/注册**：用户身份认证

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 搜索栏 | 按菜名或口味标签搜索，响应延迟≤500ms |
| 首页 | 口味标签筛选 | 点击标签快速过滤菜谱 |
| 首页 | 食谱卡片墙 | 展示所有菜谱卡片（封面图、菜名、口味标签、评分） |
| 食谱详情页 | 封面与信息 | 展示封面图、菜名、用料、口味标签 |
| 食谱详情页 | 分步展示 | 每一步配文字和步骤图片，淡入淡出切换动画 |
| 食谱详情页 | 收藏与打分 | 收藏按钮、1-5星打分组件 |
| 食谱详情页 | 相似推荐 | 基于口味标签推荐3道相似菜谱，点击跳转 |
| 发布页 | 菜谱表单 | 填写菜名、用料、做法步骤、步骤图片、封面图、口味标签 |
| 我的收藏页 | 收藏列表 | 展示已收藏菜谱卡片列表，可取消收藏 |
| 登录/注册 | 认证表单 | 用户名+密码登录或注册 |

## 3. 核心流程

**发布菜谱流程**：用户登录 → 点击"发布" → 填写菜谱信息（菜名、用料、步骤、图片、口味标签） → 提交 → 菜谱出现在首页食谱墙

**发现菜谱流程**：用户访问首页 → 浏览食谱墙/搜索/标签筛选 → 点击卡片进入详情 → 分步查看做法 → 收藏/打分 → 查看推荐菜谱并跳转

**收藏管理流程**：用户在详情页收藏 → 前往"我的收藏" → 查看收藏列表 → 可取消收藏

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：温暖橙色（#FF6B35）+ 纯白（#FFFFFF）+ 浅灰背景（#FFF8F0）
- 辅助色：深棕文字（#3D2C2E）、浅橙点缀（#FFB088）
- 按钮样式：圆角按钮，橙色主按钮带轻微阴影
- 字体：标题使用 Noto Serif SC（衬线中文），正文使用 Noto Sans SC
- 布局风格：卡片式排版，顶部固定导航栏
- 卡片效果：悬停上浮阴影 + 点击涟漪反馈
- 图标风格：线性图标，统一2px描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 搜索栏 | 圆角输入框，搜索图标，口味标签按钮组 |
| 首页 | 食谱卡片墙 | 网格布局卡片（封面图+菜名+标签+评分），悬停上浮阴影 |
| 食谱详情页 | 封面区 | 大幅封面图，菜名叠加，口味标签胶囊 |
| 食谱详情页 | 用料区 | 卡片内两列用料列表 |
| 食谱详情页 | 步骤区 | 步骤编号+文字+图片，淡入淡出切换 |
| 食谱详情页 | 操作区 | 收藏按钮（心形）、星级打分组件 |
| 食谱详情页 | 推荐区 | 3张横向小卡片，口味匹配标签 |
| 发布页 | 表单 | 分组卡片式表单，图片上传区域，标签多选 |
| 我的收藏页 | 收藏列表 | 与首页相同卡片样式，可取消收藏 |
| 导航栏 | 顶部固定 | Logo + 首页/发布/我的收藏/登录 四个导航项 |

### 4.3 响应式设计

- 桌面端优先设计，最大宽度1200px居中
- 移动端适配：卡片网格从3列→2列→1列，导航栏折叠为汉堡菜单
- 触摸优化：按钮最小44px触控区域，卡片点击区域足够大
- 页面切换流畅，FPS稳定60帧

### 4.4 动效设计

- 页面加载：卡片交错淡入（staggered reveal）
- 步骤图片切换：淡入淡出（opacity transition 300ms）
- 卡片悬停：translateY(-6px) + box-shadow加深
- 卡片点击：涟漪效果（ripple）
- 导航栏滚动：背景模糊效果（backdrop-filter: blur）
