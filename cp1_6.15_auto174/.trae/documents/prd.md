## 1. 产品概述

文学社电子书库与读者书评互动平台，旨在解决小型独立社区文学社管理员手动管理书籍入库、借阅记录和书评审核依赖纸质登记和微信群接龙、混乱且缺乏统一数据沉淀的问题。目标用户为文学社管理员和社内会员，通过统一的数字化平台实现书籍管理、借阅预约、书评互动的闭环运营。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 后台预设账号 | 书籍入库、书评审核、数据统计看板 |
| 会员 | 自助注册 | 浏览书籍、借阅预约、撰写书评、个人中心 |

### 2.2 功能模块

1. **书籍浏览与状态展示**：网格卡片展示所有书籍，含封面、书名、作者、状态标签，点击弹出详情页
2. **会员自助借阅与预约**：登录会员可借阅/预约书籍，遵守借阅规则，个人中心查看记录
3. **书评发布与审核**：会员撰写书评并评分，管理员审核后公开显示
4. **管理员书籍入库**：管理员添加新书、上传封面、自动生成二维码
5. **数据统计看板**：统计卡片+折线图展示运营数据，10秒轮询更新

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 书籍列表页 | 卡片网格 | 以200x320px圆角卡片展示书籍，含封面缩略图、书名、作者、状态标签(在馆/借出/预约中) |
| 书籍列表页 | 搜索与筛选 | 搜索框+状态下拉筛选，分页加载 |
| 书籍详情页 | 详情弹窗 | 全屏遮罩弹窗，显示完整封面、简介、藏书量、最新三条书评 |
| 书籍详情页 | 借阅/预约按钮 | 渐变按钮，借阅规则(3本/14天)，借出后可预约排队 |
| 书籍详情页 | 书评撰写 | 文本框+星级评分，提交后待审核 |
| 个人中心 | 借阅记录 | 列表展示借阅/预约记录，右侧进度条显示剩余天数 |
| 管理员看板 | 统计卡片 | 总藏书量、总会员数、本月借出量、待审核书评数 |
| 管理员看板 | 折线图 | 最近7天借阅趋势 |
| 管理员入库 | 入库表单 | 填写书名、作者、简介、ISBN、封面上传，自动生成二维码 |
| 管理员审核 | 审核列表 | 交替行背景，通过/驳回操作，驳回需填理由 |

## 3. 核心流程

**借阅流程**：会员登录 → 浏览书籍 → 点击卡片查看详情 → 点击借阅按钮 → 系统校验借阅上限(3本) → 借阅成功/提示已达上限 → 归还时按预约顺序通知

**书评流程**：会员在详情页撰写书评+评分 → 提交进入待审核 → 管理员审核列表查看 → 通过/驳回 → 通过后书评公开显示

**入库流程**：管理员填写表单+上传封面 → 系统生成二维码 → 书籍自动出现在公开书库

## 4. 用户界面设计

### 4.1 设计风格

- 主色：#667eea(紫蓝色)与#764ba2(深紫色)渐变搭配
- 背景：白色#ffffff与浅灰#f5f5f7
- 按钮风格：圆角胶囊按钮(22px圆角)，渐变背景
- 字体：Noto Sans SC为主字体，标题加粗
- 布局：顶部固定导航栏，内容区居中，卡片网格布局
- 图标：Lucide图标库

### 4.3 响应式适配

- 桌面端(≥1024px)：完整导航链接，卡片多列网格，统计卡片四列
- 平板端(768-1023px)：导航链接折叠为汉堡菜单，卡片两列，统计卡片两列布局

### 4.4 性能要求

- 首页初始加载≤2秒
- 书籍列表翻页响应≤300ms
- 书评提交后5秒内前端显示
