## 1. 产品概述

微型艺术展览策展管理系统是一款面向小型画廊和独立策展人的线上策展工具，帮助用户规划展览空间布局、管理作品与艺术家信息，并生成专业的展览日程和电子邀请函。

- 核心目标：简化策展流程，提升布展效率，让小型艺术机构也能拥有专业的数字化策展体验
- 目标用户：小型画廊主理人、独立策展人、艺术机构策展助理

## 2. 核心功能

### 2.1 用户角色

本系统为单用户角色设计，无需登录注册，所有数据本地持久化存储。

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 策展人 | 直接使用 | 作品管理、艺术家管理、展览规划、邀请函生成 |

### 2.2 功能模块

1. **展览空间规划模块**：2D网格画布、作品拖拽放置、缩放平移、面积统计
2. **作品与艺术家管理模块**：作品CRUD、艺术家信息管理、卡片展示、详情页
3. **展览日程与邀请函模块**：展览项目管理、排期冲突检测、HTML邀请函生成、打印预览

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 作品列表页 | 卡片网格、筛选侧边栏 | 作品卡片网格展示，按类别筛选，点击进入详情 |
| 作品详情页 | 详情信息、编辑入口、艺术家作品 | 展示作品完整信息，提供编辑功能，关联艺术家其他作品 |
| 展览列表页 | 展览卡片、创建入口 | 展示所有展览项目，支持创建新展览 |
| 展览详情页 | 2D画布、邀请函预览 | 组合GridCanvas和InvitationPreview，实现展览空间规划 |

## 3. 核心流程

### 3.1 策展主流程

用户首先添加作品和艺术家信息，然后创建展览项目，在2D画布上拖拽规划作品布局，最后生成电子邀请函用于分享和打印。

```mermaid
flowchart LR
    A["添加作品和艺术家"] --> B["创建展览项目"]
    B --> C["2D画布规划布局"]
    C --> D["生成电子邀请函"]
    D --> E["预览/打印/分享"]
```

### 3.2 作品管理流程

```mermaid
flowchart LR
    A["进入作品列表"] --> B["点击添加作品"]
    B --> C["填写作品信息"]
    C --> D["填写艺术家信息"]
    D --> E["保存作品"]
    E --> F["查看作品卡片"]
    F --> G["点击进入详情"]
    G --> H["编辑/删除作品"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：陶土色 #C67B5C，用于按钮、选中状态、强调元素
- **背景色**：米白色 #F5F1EB，营造温暖的艺术氛围
- **文字色**：深灰色 #333333，保证可读性
- **辅助色**：浅灰色 #E5E5E5，用于未选中状态、边框
- **按钮风格**：陶土色描边，未选中为浅灰描边，选中/悬停填充陶土色，0.3s缓动过渡
- **字体**：优雅的衬线字体展示标题，无衬线字体用于正文，体现艺术感与现代感的结合
- **布局风格**：左右两栏结构，左侧小比例侧边栏，右侧大占比内容区
- **导航栏**：极简风格，仅顶部一条2px陶土色横线分隔

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 作品列表页 | 侧边栏筛选 | 分类标签、筛选器、陶土色选中态、滑动动画 |
| 作品列表页 | 卡片网格 | 作品卡片、缩略图占位、名称/艺术家/年份、悬停缩放 |
| 作品详情页 | 详情面板 | 大缩略图、作品信息、艺术家简介、编辑按钮 |
| 作品详情页 | 关联作品 | 艺术家其他作品横向滚动列表 |
| 展览详情页 | 2D画布 | 灰色虚线网格、作品缩略图拖拽、半透明预览、缩放控制 |
| 展览详情页 | 信息面板 | 总面积、已用面积、剩余面积百分比、实时更新 |
| 邀请函页面 | 预览区域 | 展览概要、作品列表、开放时间、二维码、打印样式 |

### 4.3 交互动效

- **拖拽效果**：作品拖动时缩放1.02倍，半透明预览色块跟随鼠标
- **放置动画**：弹性缓冲动画，模拟物理落地感
- **点击反馈**：鼠标按下时缩小0.95倍，松开回弹，微小圆角
- **侧边栏**：窄屏时汉堡菜单，打开关闭滑动动画
- **过渡效果**：所有状态变化0.3s缓动过渡

### 4.4 响应式设计

- 桌面端（>768px）：左右两栏布局，侧边栏固定宽度
- 平板/移动端（≤768px）：上下堆叠布局，侧边栏收起为汉堡菜单
- 触控优化：增大点击区域，支持手势缩放平移画布

### 4.5 性能要求

- 画布拖拽和缩放响应帧率 ≥ 30FPS
- 作品列表页首次渲染时间 ≤ 1.5s
- 动画流畅，无明显卡顿
