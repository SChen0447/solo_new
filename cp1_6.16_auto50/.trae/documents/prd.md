## 1. 产品概述

交互式3D分子结构可视化应用，让科研人员和学生加载SDF格式分子数据文件，在三维空间中观察分子的原子与化学键模型，支持旋转、缩放、切换显示模式和查看分子属性信息。

- 目标用户：化学/生物领域科研人员、高校学生
- 核心价值：直观呈现分子三维结构，降低分子数据理解门槛

## 2. 核心功能

### 2.1 功能模块

1. **分子视口页**：3D分子渲染视口、文件上传区、工具栏、信息面板

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 分子视口页 | 文件上传区 | 点击上传按钮或拖拽.sdf文件（≤5MB）到指定区域，拖入时边框变蓝放大1.05倍（0.3s过渡），解析时显示进度条（模拟1-2s），成功显示分子名称和原子数，失败显示红色错误提示并自动清除 |
| 分子视口页 | 3D渲染视口 | Three.js渲染原子球体（按元素类型颜色/半径）和键圆柱体，背景渐变+星空粒子，支持球棍/线框两种模式切换（0.5s淡入淡出动画），InstancedMesh批量渲染优化 |
| 分子视口页 | 交互控制 | OrbitControls旋转/缩放/平移，缩放0.5-5倍，旋转时显示坐标轴辅助器，点击原子高亮+外发光（0.2s动画），右下角信息面板显示元素符号/编号/3D坐标/连接原子数 |
| 分子视口页 | 工具栏 | 半透明毛玻璃效果，含上传按钮、模式切换按钮、重置视角按钮，悬停亮色，点击缩小0.95动画 |

## 3. 核心流程

用户打开应用 → 拖拽/上传SDF文件 → Web Worker解析数据 → 事件总线传递数据 → 场景渲染模块构建3D模型 → 用户旋转/缩放/点击原子查看信息 → 切换显示模式

```mermaid
flowchart TD
    "A[用户上传SDF文件]" --> "B[Web Worker解析数据]"
    "B" --> "C[事件总线: molecule-parsed]"
    "C" --> "D[场景渲染模块构建3D模型]"
    "D" --> "E[用户交互: 旋转/缩放/点击]"
    "E" --> "F{操作类型}"
    "F" --> "G[切换显示模式]"
    "F" --> "H[点击原子查看信息]"
    "F" --> "I[重置视角]"
    "G --> J[事件总线: mode-changed"
    "J --> D"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝#0a0a2e → 深灰#1a1a2e渐变，文字#e0e0e0
- 按钮风格：圆角按钮，悬停#4a4a6a，点击缩小0.95动画
- 字体：JetBrains Mono（科技感），Source Sans Pro（UI）
- 布局：顶部工具栏 + 左侧信息面板 + 右侧3D视口(80%)
- 图标：lucide图标风格

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 分子视口页 | 工具栏 | 半透明毛玻璃(rgba(26,26,46,0.8), blur10px)，上传/模式切换/重置按钮 |
| 分子视口页 | 信息面板 | 左侧半透明白色圆角卡片，投影效果，显示分子信息和选中原子详情 |
| 分子视口页 | 3D视口 | 占80%宽度，渐变背景+星空粒子，球体(元素色)+圆柱体(键) |
| 分子视口页 | 拖拽区 | 虚线边框，拖入变蓝+放大，居中提示文字 |
| 分子视口页 | 进度条 | 解析时显示，模拟1-2s |

### 4.3 响应式适配

- 桌面端（≥768px）：左侧信息面板 + 右侧3D视口
- 移动端（<768px）：信息面板折叠为底部可收起抽屉（0.3s滑入动画），工具栏按钮垂直排列

### 4.4 3D场景指引

- 环境：深蓝到深灰渐变背景，半透明星空粒子装饰
- 灯光：环境光 + 平行光，标准材质光泽感
- 相机：透视相机，OrbitControls控制，缩放0.5-5倍
- 交互：旋转时显示RGB坐标轴辅助器，点击原子外发光高亮
- 原子颜色：碳#808080、氢#FFFFFF、氧#FF0000、氮#0000FF、硫#FFFF00、磷#FFA500
- 原子半径：碳1.0、氧0.8、氮0.9、氢0.6、其他0.7
- 键：半径0.2，颜色#C0C0C0
- 性能：InstancedMesh批量渲染，≥30fps@200原子
