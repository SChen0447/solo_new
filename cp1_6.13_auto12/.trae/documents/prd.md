## 1. 产品概述

音律节点（Rhythm Node）是一款基于Phaser 3引擎的网页端音乐节奏打击游戏，玩家通过点击下落音符体验打碟乐趣，配合动态粒子特效获得沉浸式节奏反馈。目标用户为休闲游戏玩家和音乐爱好者，核心价值在于精准的节奏判定机制与视觉冲击力强的粒子特效系统。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册 | 游玩游戏、查看结算、重新开始 |

### 2.2 功能模块
1. **游戏主界面**：4轨道音符下落、判定线交互、实时分数与连击显示、生命值显示
2. **结算界面**：最终分数、最高连击数、判定分布统计、重新开始按钮

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | 轨道系统 | 4条垂直轨道，音符从顶端生成匀速下落，速度分阶段递增 |
| 游戏主界面 | 判定系统 | 点击判定线区域检测命中精度（Perfect/Good/Miss），触发对应粒子特效 |
| 游戏主界面 | 分数与连击 | 实时显示分数、连击数，连击倍率计算，连击里程碑震动与闪光 |
| 游戏主界面 | 生命值 | 5颗心形图标，Miss扣1点，归零触发游戏结束 |
| 游戏主界面 | 粒子特效 | Perfect金色星形爆发+判定线闪烁、Good白色圆形粒子、Miss红色碎片飘散 |
| 结算界面 | 结算面板 | 半透明模糊背景，显示最终分数、最高连击、Perfect/Good/Miss数量 |
| 结算界面 | 重新开始按钮 | 悬停放大1.1倍+颜色渐变，点击重置所有游戏状态 |

## 3. 核心流程

玩家进入游戏后，BootScene预加载资源，完成后自动切换至GameScene。音符按内置节奏数据从4条轨道顶端生成并下落，玩家在判定线区域点击匹配音符。命中精度决定粒子特效与分数变化，Miss扣减生命值。生命值归零或歌曲结束进入结算界面，玩家可查看统计并重新开始。

```mermaid
flowchart TD
    "启动游戏" --> "BootScene资源预加载"
    "BootScene资源预加载" --> "GameScene游戏主循环"
    "GameScene游戏主循环" --> "音符生成与下落"
    "音符生成与下落" --> "玩家点击交互"
    "玩家点击交互" --> "精度判定"
    "精度判定" --> "Perfect/Good: 加分+粒子特效"
    "精度判定" --> "Miss: 扣生命+碎片特效"
    "Perfect/Good: 加分+粒子特效" --> "更新分数与连击"
    "Miss: 扣生命+碎片特效" --> "更新分数与连击"
    "更新分数与连击" --> "生命值为0或歌曲结束?"
    "生命值为0或歌曲结束?" -- 否 --> "音符生成与下落"
    "生命值为0或歌曲结束?" -- 是 --> "结算界面"
    "结算界面" --> "重新开始"
    "重新开始" --> "GameScene游戏主循环"
```

## 4. 界面设计

### 4.1 设计风格
- 主色调：深蓝渐变至纯黑背景，轨道分隔半透明白色虚线，判定线亮白色发光
- 音符颜色：红/蓝/绿/紫四轨道对应色，圆形主体带渐变外发光
- 字体：像素风格字体 Press Start 2P，用于分数、连击数显示
- 按钮风格：圆角按钮，悬停放大1.1倍并颜色渐变
- 布局风格：全屏纵向4轨道居中布局，顶部HUD信息栏
- 粒子风格：星形（Perfect金色）、圆形（Good白色）、碎片（Miss红色）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 背景层 | 深蓝到纯黑纵向渐变 |
| 游戏主界面 | 轨道层 | 4条等宽轨道，半透明白色虚线分隔 |
| 游戏主界面 | 判定线 | 底部亮白色发光横线 |
| 游戏主界面 | 音符 | 圆形+轨道色外发光，下落动画 |
| 游戏主界面 | HUD | 顶部左侧分数、中间连击数、右侧心形生命值 |
| 游戏主界面 | 闪光效果 | 白色半透明覆盖层渐隐（连击里程碑） |
| 结算界面 | 面板 | 半透明模糊背景居中卡片 |
| 结算界面 | 统计文字 | 分数、最高连击、判定分布 |
| 结算界面 | 按钮 | 重新开始按钮，悬停放大+渐变 |

### 4.3 响应式适配
- 桌面优先设计，全屏适配
- 移动端（宽度<768px）自动缩放轨道和音符比例，保证触控区域足够大
- 使用Phaser的scale管理器实现自适应

### 4.4 性能指标
- 全程60fps稳定运行
- 粒子峰值不超过200个，超出自动回收旧粒子
- 音符对象池化回收，避免频繁GC
