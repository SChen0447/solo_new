## 1. 产品概述

社区咖啡馆顾客在线竞拍每周限定特调咖啡的互动平台，让用户浏览每周限定特调列表、参与实时竞拍出价，拍卖结束后系统自动判定赢家并生成兑换券码。目标用户为社区咖啡馆的忠实顾客和咖啡爱好者，通过竞拍机制提升特调咖啡的稀缺感和互动趣味性。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 邮箱+用户名注册 | 浏览拍卖、参与出价、查看历史记录和兑换券 |

### 2.2 功能模块
1. **拍卖列表页**：展示每周限定特调咖啡卡片（最多6款），卡片显示咖啡名称、烘焙程度、当前出价、剩余时间，颜色按烘焙程度编码
2. **拍卖详情页**：咖啡信息展示、实时价格曲线图、出价历史列表、倒计时环形进度条、出价表单
3. **用户认证页**：注册与登录，邮箱格式验证，用户名2-12字符
4. **个人中心页**：历史出价记录、获胜记录、兑换券码查看与复制

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 拍卖列表页 | 咖啡卡片列表 | 展示所有有效拍卖，两列网格布局，卡片含咖啡名称/烘焙程度/当前出价/倒计时，颜色编码：浅烘焙#f5e6c8、中烘焙#e8c8a0、深烘焙#b8855a |
| 拍卖详情页 | 咖啡信息区 | 顶部高清渲染图区域（渐变色占位#c7a06c→#8b5e3c），下方三栏布局 |
| 拍卖详情页 | 出价历史列表 | 左侧，每条记录带时间戳和用户名，用户自己出价高亮#f5d6a8 |
| 拍卖详情页 | 实时价格曲线 | 中间，SVG折线图，数据点悬停显示金额和出价者 |
| 拍卖详情页 | 倒计时与出价 | 右侧，环形进度条（48px直径，黑色圆弧，<60秒变红#ff4444+脉冲动画），出价按钮56px圆形#8b5e3c |
| 用户认证页 | 注册表单 | 邮箱+用户名+密码注册，邮箱格式验证，用户名2-12字符 |
| 用户认证页 | 登录表单 | 邮箱+密码登录 |
| 个人中心页 | 历史记录 | 所有参与过的拍卖、出价记录和是否获胜 |
| 个人中心页 | 兑换券码 | 8位字母数字混合券码，点击复制按钮显示确认toast（2秒消失） |

## 3. 核心流程

用户注册/登录 → 浏览拍卖列表 → 点击卡片进入详情页 → 查看实时出价和倒计时 → 输入出价金额（必须高于当前出价+5元最小加价）→ WebSocket提交出价 → 所有在线用户同步更新 → 倒计时归零自动锁定 → 系统判定赢家 → 生成8位兑换券码 → 获胜用户在个人中心查看券码

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：米白背景#faf0e6，主色咖啡棕#8b5e3c，深棕#6a4320
- **辅助色**：浅烘焙淡黄#f5e6c8、中烘焙浅橙#e8c8a0、深烘焙深棕#b8855a
- **按钮风格**：出价按钮圆形56px，主色#8b5e3c，悬停#6a4320，成功闪烁绿光0.5秒
- **字体**：标题使用衬线体营造旧报纸风格，正文使用易读的无衬线体
- **布局风格**：卡片式布局，柔和阴影0 4px 12px rgba(0,0,0,0.08)，圆角12px
- **装饰风格**：手写咖啡杯元素与旧报纸纹理，暖色调复古感

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 拍卖列表页 | 咖啡卡片 | 卡片阴影+圆角12px，烘焙程度颜色编码，两列网格 |
| 拍卖详情页 | 顶部图片区 | 渐变色占位#c7a06c→#8b5e3c |
| 拍卖详情页 | 三栏布局 | 左-出价历史、中-价格曲线、右-倒计时+出价 |
| 拍卖详情页 | 环形倒计时 | 48px直径黑色圆弧，<60秒变红+脉冲 |
| 拍卖详情页 | 出价按钮 | 圆形56px，#8b5e3c，悬停#6a4320 |
| 拍卖详情页 | 闪光效果 | 出价成功顶部橙色闪光0.3秒 |
| 拍卖详情页 | 淡入动画 | 新出价条目0.5秒淡入 |
| 错误模态框 | 断网提示 | 半透明遮罩，圆角16px，红色边框#e74c3c |

### 4.3 响应式适配
- 桌面端：拍卖列表两列，详情页三栏布局
- 平板端：拍卖列表一列，详情页两栏布局
- 移动端：单列布局，详情页单栏垂直排列

### 4.4 动画效果
- 出价成功：页面顶部橙色闪光0.3秒
- 新出价条目：淡入动画0.5秒
- 出价按钮成功：闪烁绿光0.5秒
- 倒计时<60秒：1秒缩放脉冲动画
- 兑换券复制：确认toast 2秒后消失
