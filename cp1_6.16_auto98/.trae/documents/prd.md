## 1. 产品概述
团队知识库文档协作与版本对比应用，面向团队协作场景，支持多人协同编辑文档、自动记录版本快照、可视化对比不同版本差异，并支持版本回滚操作。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 团队成员 | 模拟登录 | 编辑文档、查看版本历史、对比版本、回滚版本 |

### 2.2 功能模块
1. **文档编辑页**: 富文本编辑器、自动保存、协作状态指示
2. **版本历史侧栏**: 版本列表、滚动加载、版本选中
3. **版本对比覆盖层**: 差异高亮对比、回滚操作、确认对话框

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 文档编辑页 | 富文本编辑器 | contenteditable实现，5秒无活动自动保存，保存提示淡入淡出 |
| 文档编辑页 | 协作状态指示 | 在线人数头像、编辑闪烁动画、最后保存时间 |
| 版本历史侧栏 | 版本列表 | 版本序号+时间戳，选中高亮，滚动加载(初始20条) |
| 版本历史侧栏 | 对比按钮 | 圆形按钮，点击打开半屏差异对比覆盖层 |
| 版本对比覆盖层 | 差异对比视图 | 左右分栏对比，新增/删除/修改行高亮 |
| 版本对比覆盖层 | 回滚操作 | 回滚按钮+确认对话框+滑入通知条 |

## 4. 用户界面设计

### 4.1 设计风格
- 主色调: 蓝色 #1976D2，浅色背景 #BBDEFB，强调色 #FF5722
- 按钮风格: 无边框扁平设计，圆角6px，hover时0.2秒背景色过渡
- 字体: 系统字体栈，编辑器14px/行高1.6
- 布局风格: 左右分栏(280px + 剩余宽度)
- 交互反馈: 按下时scale(0.98)，0.1秒缓动

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 文档编辑页 | 编辑器区域 | 浅灰底色#F5F5F5，文字#333，14px，行高1.6，段落间距12px |
| 文档编辑页 | 协作状态栏 | 圆形头像32px(多色)，闪烁黄色边框脉冲动画 |
| 文档编辑页 | 保存提示 | 淡入"已保存"文字，2秒后淡出 |
| 文档编辑页 | 最后保存时间 | 右下角，灰色#9E9E9E，12px |
| 版本历史侧栏 | 列表项 | 版本序号+时间，选中#E3F2FD，hover #BBDEFB |
| 版本对比覆盖层 | 对比视图 | 左侧旧版本#FFEBEE，右侧新版本#E8F5E9 |
| 版本对比覆盖层 | 差异高亮 | 新增#C8E6C9，删除#FFCDD2，修改#FFF9C4 |
| 版本对比覆盖层 | 回滚按钮 | 红色#D32F2F，hover变暗 |
| 版本对比覆盖层 | 确认对话框 | 半透明遮罩，居中，圆角8px，阴影 |
| 版本对比覆盖层 | 通知条 | 绿色#4CAF50，底部滑入，3秒消失 |

### 4.3 响应式适配
- 桌面优先设计，左右分栏布局
- 屏幕宽度 < 768px: 左侧版本列表变为可折叠侧边栏(展开占全屏高度一半，折叠只显示图标)
- 移动端差异对比改为全屏覆盖
