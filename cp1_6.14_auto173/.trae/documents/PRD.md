## 1. 产品概述

在线虚拟试妆镜是一款基于Web的美妆试色应用，用户上传自拍照后即可实时体验不同品牌和色系的口红、眼影、腮红妆容效果。

- **核心功能**：基于面部关键点检测的虚拟妆容叠加、化妆品色板选择、妆容浓度调节、试妆效果保存与分享
- **目标用户**：美妆爱好者、化妆品消费者、社交分享用户
- **产品价值**：足不出户即可预览真实妆容效果，降低试妆成本，辅助购买决策

## 2. 核心功能

### 2.1 功能模块

1. **面部检测与妆容渲染模块**：上传照片、关键点自动检测、妆容实时叠加渲染
2. **化妆品选择面板**：口红/眼影/腮红三分类、每类6+色号、选中状态反馈
3. **浓度调节模块**：透明度滑块、百分比显示、数字动画
4. **相册管理模块**：试妆效果保存、缩略图列表展示、历史记录加载
5. **分享模块**：复制链接、社交媒体分享、动画弹出面板

## 3. 核心流程

用户上传自拍照 → 系统自动检测面部关键点（1秒内完成，带旋转进度条）→ 用户选择化妆品类别与颜色 → 实时Canvas渲染妆容叠加 → 调节浓度滑块微调 → 点击保存按钮 → 缩略图加入相册列表 → 点击分享按钮选择分享方式

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：品牌渐变 #667EEA → #764BA2（蓝紫渐变）
- **背景色**：#F5F5F7（浅灰白）
- **面板背景**：纯白色，卡片阴影 box-shadow: 0 2px 8px rgba(0,0,0,0.08)
- **关键点标记**：直径4px半透明渐变青色小圆圈
- **色号色块**：直径32px圆形，1px浅灰边框，悬停1.1倍放大+光晕
- **按钮风格**：圆形主按钮，渐变蓝紫色，悬停1.05倍+扩散光晕，按压回弹
- **缩略图**：80x80px，圆角8px，悬停1.1倍+加深阴影

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素与动画 |
|-----------|-------------|-------------|
| 主页 | 检测进度条 | 旋转圆形加载动画，1秒内完成 |
| 主页 | 色号选中效果 | 0.5秒从内到外扩散脉动高亮环 |
| 主页 | 滑块数值 | 百分比显示，0.2秒数字递增/递减动画 |
| 主页 | 保存按钮 | 悬停1.05倍+光晕，点击缩小95%回弹 |
| 主页 | 相册移除动画 | 向左滑动淡出，0.3秒 |
| 主页 | 分享面板 | 从底部向上滑入，0.3秒 ease-out |

### 4.3 响应式设计

- **桌面端**：最小宽度1024px，三栏布局（左侧色板+中间画布+右侧相册）
- **平板端**：左右面板折叠至底部，画布居中展示
- **移动端**：纵向堆叠布局，画布优先展示
