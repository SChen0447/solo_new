## 1. 产品概述

在线协作食谱编辑与分享平台，让用户可以浏览、创建和编辑数字食谱，支持多人实时协作编辑，自动同步更改。面向美食爱好者、家庭厨师和烹饪社区。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 用户名注册登录 | 浏览食谱、创建/编辑自己的食谱、收藏和评分 |
| 协作者 | 被邀请进入编辑 | 实时编辑指定食谱、查看在线协作者 |

### 2.2 功能模块

1. **首页（食谱浏览器）**: 食谱卡片列表、标签和用料搜索、封面图/渐变占位
2. **食谱详情页**: 完整食谱内容展示、评分点赞、收藏功能
3. **食谱编辑器**: 三栏编辑器（配料列表、烹饪步骤、辅助面板）、实时协作同步
4. **个人中心**: 我的食谱、收藏食谱

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 食谱卡片列表 | 展示食谱卡片，包含封面图（或渐变色占位）、标题、作者、难度和预估时长，支持按标签和用料搜索过滤 |
| 首页 | 搜索栏 | 按标签和用料关键词搜索食谱，响应时间低于500ms |
| 食谱详情页 | 食谱内容 | 展示配料列表、烹饪步骤、营养信息 |
| 食谱详情页 | 评分与收藏 | 1-5星评分（显示平均分和人数）、点赞、收藏 |
| 食谱编辑器 | 左侧配料面板 | 增删改配料，自动计算总卡路里和营养素 |
| 食谱编辑器 | 中间步骤区 | 每个步骤含图片和文字说明，可拖拽调整顺序 |
| 食谱编辑器 | 右侧辅助面板 | 可启动/暂停/重置的计时器，预设5分/10分/30分 |
| 食谱编辑器 | 协作功能 | WebSocket实时同步，在线协作者头像显示在右上角 |
| 个人中心 | 我的食谱 | 展示用户创建的食谱列表 |
| 个人中心 | 收藏食谱 | 展示用户收藏的食谱列表 |

## 3. 核心流程

用户注册登录 → 浏览首页食谱列表 → 点击卡片查看详情 → 点击创建/编辑进入编辑器 → 左侧配料+中间步骤+右侧计时器三栏协作 → 通过WebSocket实时同步到其他协作者 → 发布食谱 → 其他用户浏览、评分、收藏

## 4. 用户界面设计

### 4.1 设计风格

- 主色：番茄红 #e74c3c，暗红 #c0392b（悬停态）
- 辅色：奶白色 #fdf5e6（背景）
- 卡片圆角 16px，微弱阴影，悬停时阴影加深且轻微上浮
- 按钮使用主色填充，悬停变暗红 #c0392b
- 所有交互 0.2-0.3秒平滑过渡动画
- 字体：标题使用 Playfair Display，正文使用 Noto Sans SC
- 布局：卡片式布局，顶部导航栏

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 导航栏 | Logo + 搜索框 + 创建按钮 + 用户头像，背景奶白，Logo番茄红 |
| 首页 | 食谱卡片 | 封面图/渐变占位 + 标题 + 作者 + 难度标签 + 时长，16px圆角，悬停上浮 |
| 首页 | 搜索过滤 | 标签chips + 用料输入框，实时过滤 |
| 食谱详情页 | 头部 | 大封面图 + 标题 + 作者信息 + 评分星星 + 收藏按钮 |
| 食谱详情页 | 配料区 | 左侧配料卡片，含营养素汇总 |
| 食谱详情页 | 步骤区 | 竖向步骤列表，每步含编号、图片、说明 |
| 编辑器 | 左侧配料面板 | 配料列表 + 添加按钮 + 营养素统计 |
| 编辑器 | 中间步骤区 | 步骤卡片列表 + 拖拽手柄 + 图片上传 |
| 编辑器 | 右侧面板 | 计时器（数字显示+控制按钮+预设）+ 协作者头像 |
| 个人中心 | 标签切换 | 我的食谱 / 收藏食谱 切换标签 |
| 个人中心 | 食谱列表 | 紧凑型食谱卡片网格 |

### 4.3 响应式适配

- 桌面端优先，编辑器三栏布局
- 平板端编辑器改为两栏（配料+步骤合并，辅助面板独立）
- 移动端单栏布局，底部Tab导航，编辑器面板可滑动切换
- 触控优化：按钮最小44px，拖拽支持触摸

## 5. 性能要求

- 食谱列表首次加载和搜索响应时间 < 500ms
- 编辑器实时同步延迟 < 200ms
- 卡片列表虚拟滚动（大量食谱时）
- WebSocket消息节流优化
