# 数字电路模拟沙盒 - 产品需求文档

## 1. 项目概述

### 1.1 项目背景
在学习数字电路时，学生往往难以直观理解逻辑门组合后的信号传递效果。本应用提供一个交互式的浏览器沙盒环境，让用户可以通过拖拽逻辑门元件、连接引脚、实时观察信号传递，从而更直观地学习和理解数字电路原理。

### 1.2 目标用户
- 电子工程专业学生
- 计算机科学初学者
- 数字电路爱好者

### 1.3 核心价值
- **可视化学习**：将抽象的逻辑运算转化为直观的图形交互
- **实时反馈**：信号传递延迟不超过100ms，即时看到电路效果
- **低门槛**：无需安装软件，浏览器即可使用
- **高性能**：支持20+元件和30+连线同时运行，保持60fps流畅度

## 2. 功能需求

### 2.1 逻辑门元件库
| 元件类型 | 描述 | 输入引脚 | 输出引脚 |
|---------|------|---------|---------|
| AND | 与门，全1出1 | 2 | 1 |
| OR | 或门，有1出1 | 2 | 1 |
| NOT | 非门，取反 | 1 | 1 |
| XOR | 异或门，不同出1 | 2 | 1 |
| INPUT | 输入按钮，可切换高/低电平 | 0 | 1 |
| OUTPUT | 输出指示器，显示电平状态 | 1 | 0 |

### 2.2 交互功能

#### 2.2.1 元件拖拽
- 从右侧边栏拖拽元件到画布区域
- 拖拽过程中有弹性吸附动画效果
- 元件放置后显示清晰的输入/输出引脚端点

#### 2.2.2 连线功能
- 从输出引脚按下鼠标拖拽到输入引脚完成连线
- 连线采用贝塞尔曲线路径自动生成
- 连线过程中随鼠标移动实时重绘
- 连接完成后线体带有半透明发光效果
- 连线带有脉冲动画指示信号流动方向

#### 2.2.3 信号传递
- 点击 INPUT 元件切换高/低电平
- 信号沿连线自动传递，延迟不超过100ms
- OUTPUT 元件实时显示电平状态：红色高亮=高电平，灰色=低电平

#### 2.2.4 框选与操作
- 鼠标拖拽绘制半透明蓝色矩形选区
- 选中的元件边框变为蓝色并轻微上浮
- 支持整体移动选中元件
- 按 Delete 键删除选中元件和相关连线

### 2.3 视觉设计

#### 2.3.1 整体风格
- **深色主题**：背景色 #1a1a2e
- **磨砂玻璃效果**：元件使用半透明背景 + backdrop-filter 模糊
- **辅助网格**：虚线网格作为参考线
- **过渡动画**：所有交互使用 300ms ease-out 平滑过渡

#### 2.3.2 布局
- 右侧边栏固定宽度 320px，包含元件库
- 画布区域占据剩余空间
- 画布支持无限滚动/平移

---

## 3. 非功能需求

### 3.1 性能要求
- 画布同时容纳 20 个元件和 30 条连线时保持 60fps
- 信号传递延迟不超过 100ms
- 拖拽操作响应时间 < 16ms

### 3.2 技术约束
- 使用 TypeScript 编写
- 原生 Canvas API 渲染，不依赖框架
- Vite 构建工具
- 支持现代浏览器（Chrome、Firefox、Safari 最新版本）
