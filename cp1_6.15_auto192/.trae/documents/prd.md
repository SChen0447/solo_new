## 1. 产品概述

为小型独立书店设计的个性化书单推荐与社区书评互动应用。书店管理员可创建主题书单（如夏日推理精选），读者可浏览书单、对书籍评分和撰写短评，系统根据评分和阅读历史为每位读者生成个性化推荐。

- 目标用户：小型独立书店的管理员和读者
- 核心价值：连接书店与读者，通过个性化推荐提升阅读体验，通过书单和评论增强社区互动

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 本地Token认证 | 创建/编辑书单、添加书籍到书单 |
| 普通读者 | 默认用户(id=1) | 浏览书单、评分、写短评、查看推荐 |

### 2.2 功能模块

1. **首页**：书单卡片网格展示、搜索筛选、个性化推荐面板、创建书单入口
2. **书单详情页**：书单描述、书籍卡片列表、排序功能
3. **个人中心页**：评分评论历史、统计数据

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 书单卡片网格 | 以CSS Grid网格展示所有主题书单，每个卡片含封面、标题、书籍数量和平均评分（半星精确），悬停上移6px并扩大阴影，点击进入详情页 |
| 首页 | 搜索筛选 | 搜索框（占位文本"搜索书单或书籍"）模糊匹配书单标题或书籍名称 |
| 首页 | 推荐面板 | 右侧固定宽度320px深灰面板，展示4本推荐书籍，每本有"不喜欢"按钮，点击后翻转动画消失并重新推荐 |
| 首页 | 创建书单 | 管理员专属绿色按钮，弹出模态框输入标题、描述和多选书籍 |
| 书单详情页 | 书单信息 | 顶部显示书单描述和创建日期 |
| 书单详情页 | 书籍列表 | 卡片列表展示所有书籍，支持按评分或添加时间排序 |
| 书单详情页 | 评分与短评 | 每本书卡片下方1-5星评分（金色#ffc107），"写短评"按钮展开140字文本框 |
| 个人中心页 | 评分历史 | 按时间倒序展示所有评分评论条目 |
| 个人中心页 | 统计区域 | 总评分数、平均评分、最常评分作者前3名 |

## 3. 核心流程

**读者浏览与互动流程**：用户进入首页浏览书单 → 点击书单进入详情页 → 浏览书中并评分 → 撰写短评 → 系统根据评分历史生成个性化推荐 → 用户在首页右侧查看推荐

**管理员创建书单流程**：管理员登录 → 点击"创建书单"按钮 → 填写标题、描述 → 多选书籍 → 提交 → 新书单出现在首页顶部并带"新"标签

## 4. 用户界面设计

### 4.1 设计风格

- 主题：深色暗夜书馆风格，温暖而沉浸
- 主背景色：#121212，侧边栏：#1e1e2e，卡片：#2a2a2a
- 文字主色：#e0e0e0，强调色：金色#ffc107（评分）、绿色#4caf50（创建）、橙色#ff9800（新标签）、红色#e53935（不喜欢）
- 按钮风格：圆角矩形，内边距适中，悬停变色/变暗
- 字体：思源宋体/Playfair Display（标题）+ Noto Sans SC（正文）
- 布局：CSS Grid卡片网格，最大宽度1200px居中，minmax(280px, 1fr)自动填充
- 动画：悬停上移6px+阴影扩大(0.3s ease)，渐入动画(opacity+translateY 0.4s)，翻转动画(360度Y轴0.6s)，模态框backdrop-filter blur(6px)

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 导航栏 | 固定顶部60px，半透明背景rgba(18,18,18,0.95)，底部1px #333分隔线，左侧应用名，右侧"首页"和"个人中心"链接（当前页下划线动画0→100% 0.3s） |
| 首页 | 书单网格 | CSS Grid，max-width 1200px居中，gap 16px，卡片圆角10px，渐入动画按索引延迟0.1s |
| 首页 | 搜索框 | 居中搜索框，占位文本"搜索书单或书籍"，深色输入框样式 |
| 首页 | 推荐面板 | 固定宽度320px，背景#1e1e2e，顶部渐变磨砂rgba(255,255,255,0.05)，推荐卡片带翻转动画 |
| 首页 | 创建书单按钮 | 绿色#4caf50，圆角8px，内边距10px 20px，管理员登录后显示 |
| 书单详情页 | 书单头部 | 描述文本+创建日期，深色卡片背景 |
| 书单详情页 | 排序控件 | 按评分/添加时间排序切换 |
| 书单详情页 | 书籍卡片 | 封面图+书名+作者+星级评分+短评输入框 |
| 个人中心页 | 历史列表 | 60x60封面小图+书名+星级+短评+时间，按时间倒序 |
| 个人中心页 | 统计卡片 | 浅灰#f5f5f5背景，圆角8px，内边距12px |

### 4.3 响应式设计

- 桌面优先设计，移动端自适应
- 宽度<768px时：网格改为单列，推荐面板变为底部展示
- 宽度<768px时：导航栏变为汉堡菜单，三条横线图标，点击展开侧滑菜单(宽200px，背景#1e1e2e)
- min-width 320px支持
