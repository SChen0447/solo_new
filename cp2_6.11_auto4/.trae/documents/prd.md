## 1. 产品概述

摄影作品集线上画廊展示页面，让摄影师或设计师上传一组照片后生成精美的线上画廊。面向摄影师、设计师及视觉创作者，提供优雅、流畅的作品展示体验。

## 2. 核心功能

### 2.1 功能模块

1. **画廊主页**：网格视图展示、分类筛选、灯箱大图、收藏交互

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 画廊主页 | 网格视图 | 默认以网格展示所有照片缩略图，悬停时向上浮起并带阴影过渡动画 |
| 画廊主页 | 灯箱模式 | 点击缩略图进入全屏灯箱，照片以淡入方式大图展示，支持左右键盘箭头切换 |
| 画廊主页 | 分类筛选栏 | 包含"全部"、"风景"、"人像"、"街拍"等标签按钮，点击后只显示对应分类照片，切换时照片列表交错飞入动画出现，按钮选中状态有下划线滑动效果 |
| 画廊主页 | 收藏按钮 | 每张照片右下角心形收藏按钮，点击时心形图标弹出并填充红色，伴有微小爆炸粒子特效，收藏数量在页面左上角实时累加显示 |

## 3. 核心流程

用户打开页面 → 查看网格视图中的照片缩略图 → 可通过分类筛选栏切换查看不同类别 → 悬停缩略图查看浮起效果 → 点击缩略图进入灯箱大图模式 → 键盘左右箭头切换照片 → 点击心形按钮收藏照片 → 页面左上角实时显示收藏数量

## 4. 用户界面设计

### 4.1 设计风格

- 主色：纯白底色（#FFFFFF）搭配深灰文字（#2D2D2D）
- 点缀色：暖木色（#C4956A）用于按钮、下划线等强调元素
- 按钮风格：圆角标签按钮，选中状态带暖木色下划线滑动效果
- 字体：标题使用 Playfair Display 衬线字体，正文使用 DM Sans 无衬线字体
- 布局：顶部导航栏（毛玻璃效果）+ 分类筛选 + 网格照片区
- 图标：心形SVG图标，使用内联SVG实现粒子特效

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 画廊主页 | 导航栏 | 半透明毛玻璃背景、左上角收藏计数、居中标题 |
| 画廊主页 | 分类筛选栏 | 水平标签按钮组、下划线滑动指示器 |
| 画廊主页 | 网格视图 | 响应式网格、照片卡片带圆角、悬停浮起阴影 |
| 画廊主页 | 灯箱模式 | 全屏黑色遮罩、居中大图、左右切换箭头、关闭按钮 |
| 画廊主页 | 收藏按钮 | 心形SVG图标、点击弹出动画、红色填充、粒子爆炸 |

### 4.3 响应式设计

- 桌面优先设计，移动端自适应
- 桌面：4列网格，照片间距24px
- 平板：3列网格，照片间距16px
- 手机：2列网格，照片间距8px，导航栏压缩，紧凑排列
- 所有交互反馈在300ms内完成
- 整体动画帧率保持在60fps

### 4.4 图片资源

- 使用 SDXL API 生成占位图片，12+张不同分类的照片
- 图片尺寸：缩略图使用 landscape_4_3，灯箱大图使用 landscape_16_9
