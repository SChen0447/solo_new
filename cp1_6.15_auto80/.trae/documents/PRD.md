## 1. 产品概述

在线编程作业批改与反馈协作平台，为教师提供编程题目创建、学生代码行级批注与统计功能，为学生提供代码提交、自动测试与反馈查看功能，实现编程教学的全流程数字化协作。

- 核心价值：将编程作业的出题、提交、自动测试、人工批注整合为一体化平台，提升教学效率与反馈质量
- 目标用户：高校/培训机构的编程课程教师和学生

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 教师 | 平台分配账号 | 创建/编辑/删除题目、查看学生提交、添加行级批注、查看统计 |
| 学生 | 平台分配账号 | 查看题目、提交代码、查看测试结果与批注反馈、查看提交历史 |

### 2.2 功能模块

1. **教师面板**：题目卡片列表、创建/编辑题目、提交统计表、查看学生代码与批注
2. **学生提交页面**：题目详情、代码编辑器、自动测试结果展示
3. **批注页面**：代码行级批注、批注气泡与侧边列表
4. **学生历史面板**：按题目分组的提交历史、最近提交状态

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 教师面板 | 题目卡片列表 | 以卡片形式展示所有题目，显示标题、难度标签，支持点击编辑/删除 |
| 教师面板 | 创建/编辑题目 | 表单包含标题(max40字符)、描述(Markdown)、示例输入输出(至多3组)、难度等级 |
| 教师面板 | 提交统计表 | 列出每题总提交次数、通过次数、通过率，低于50%橙色高亮 |
| 教师面板 | 查看提交与批注 | 左栏代码(6)、右栏批注列表与测试结果(4)，支持行级批注 |
| 学生提交页面 | 题目详情 | 展示当前题目标题、描述、示例输入输出 |
| 学生提交页面 | 代码编辑器 | textarea增强，monospace字体14px、行号、语法高亮(JS/Python) |
| 学生提交页面 | 测试结果 | 用例卡片展示通过/失败，显示输入/期望输出/实际输出 |
| 学生历史面板 | 提交历史列表 | 按题目分组，显示最近提交状态标签(绿通过/红失败)和提交时间 |

## 3. 核心流程

### 教师出题流程
教师登录 → 进入教师面板 → 点击创建题目 → 填写标题/描述/示例/难度 → 保存 → 题目出现在卡片列表

### 学生提交流程
学生登录 → 选择题目 → 在代码编辑器编写代码 → 点击提交 → 后端运行测试 → 返回测试结果 → 显示通过数/总数

### 教师批注流程
教师查看学生提交 → 点击代码行号旁区域 → 输入批注 → 回车提交 → 批注气泡显示在代码行右侧 → 批注列表同步更新

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深色侧边栏 #1a1a2e，主内容区背景 #f8f9fa
- 强调色：按钮渐变 #667eea → #764ba2
- 难度标签色彩：入门 #e8f5e9、进阶 #fff3e0、高手 #fce4ec
- 字体：代码区 monospace 14px，界面使用系统字体栈
- 布局：左240px深色侧边栏 + 右侧主内容区，CSS Grid/Flexbox两列自适应
- 卡片风格：白底圆角12px，阴影0 2px 8px rgba(0,0,0,0.08)
- 按钮：圆角8px渐变，悬停阴影加深，0.3s过渡，点击scale动画0.2s

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 教师面板 | 题目卡片 | 宽320px，白底#ffffff，圆角12px，阴影，标题#1a1a2e，难度标签圆角16px |
| 教师面板 | 统计表 | 表格布局，通过率<50%橙色高亮 |
| 学生提交 | 代码编辑器 | textarea，monospace 14px，行号显示，6:4左右分栏 |
| 学生提交 | 提交按钮 | 宽120px，圆角8px，渐变#667eea→#764ba2 |
| 学生提交 | 测试结果 | 用例卡片内边距12px，圆角8px，通过#e8f5e9/失败#ffebee |
| 批注页面 | 批注输入框 | 宽300px，背景#f5f5f5，圆角8px，边框1px solid #e0e0e0 |
| 批注页面 | 批注气泡 | 圆角12px，填充#fff3e0，最大宽400px，三角箭头 |
| 全局 | 头部 | fixed，高60px，白底，底部阴影，左侧平台名，右侧圆形40px头像 |
| 全局 | 侧边栏 | 宽240px，背景#1a1a2e，白色文字，悬停#16213e |

### 4.3 响应式

- 桌面优先设计，最小宽度320px卡片，最大1200px居中
- CSS Grid自适应两列布局
- 代码查看页面左右6:4分栏
