## 1. 产品概述

魔法瓶精灵（Bottle Spirits）是一款浏览器端精灵收集与培育互动应用。玩家在独特的魔法瓶中收集、培育五种元素小精灵，通过喂食、播放音乐和清洁瓶子提升好感度，解锁瓶身装饰和互动动作，打造属于自己的精灵收藏。

- 目标用户：喜爱治愈系、收集类休闲游戏的玩家
- 核心价值：提供沉浸式的魔法瓶精灵培育体验，融合视觉装饰与互动反馈

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册，本地存储 | 培育精灵、装饰瓶子、查看图鉴、每日签到 |

### 2.2 功能模块

1. **精灵交互页**：瓶子3D感展示、精灵动画、喂食/音乐/清洁操作面板
2. **收藏图鉴页**：精灵卡片网格、种类筛选、等级排序、进入精灵管理
3. **装饰管理弹窗**：瓶塞/瓶内背景/光效装饰更换、材料消耗提示
4. **每日签到面板**：月历签到、连续签到奖励、材料收集
5. **导航栏**：五个模块图标切换，收起/展开动画

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 精灵交互页 | 瓶子可视化 | 中央展示魔法瓶，支持360度旋转查看，瓶身装饰随精灵属性变化 |
| 精灵交互页 | 精灵动画反馈 | 喂食/音乐/清洁后精灵做出对应CSS关键帧动画，表情气泡弹缩显示 |
| 精灵交互页 | 操作面板 | 三个喂食按钮、三个音乐选择、清洁按钮，1.5秒冷却动画 |
| 精灵交互页 | 心情衰减 | 每30秒自动计算心情衰减，长时间无互动心情下降 |
| 收藏图鉴页 | 卡片网格 | 显示精灵剪影/等级/名字，未解锁灰色问号带闪烁动画 |
| 收藏图鉴页 | 筛选排序 | 按五种元素筛选，按等级降序排序，交错淡入动画 |
| 装饰管理弹窗 | 装饰更换 | 三类装饰位展示与替换，过渡动画0.6秒，材料消耗提示 |
| 装饰管理弹窗 | 材料系统 | 材料通过签到和互动获取，装饰消耗特定材料 |
| 每日签到面板 | 日历签到 | 月历展示，已签日期金色星标，7天周期进度条 |
| 每日签到面板 | 奖励递增 | 第1天10材料、第3天水晶塞、第5天星光露×2、第7天解锁背景物 |

## 3. 核心流程

玩家进入应用后，通过左侧导航栏切换五个模块。主流程如下：

1. 玩家在精灵交互页选择一只精灵，查看其瓶子外观和当前状态
2. 通过喂食、播放音乐、清洁操作提升好感度和经验值
3. 积累材料后进入装饰管理弹窗，更换瓶塞/背景/光效装饰
4. 在收藏图鉴页查看所有精灵，筛选和排序管理
5. 每日签到获取材料，连续签到获得递增奖励

## 4. 用户界面设计

### 4.1 设计风格

- 主色：深靛蓝到墨黑的径向渐变背景，暖金色（#D4AF37→#B8860B）作为强调色
- 按钮风格：圆角磨砂玻璃按钮，主要操作暖金渐变，危险操作暗红色
- 字体：Playfair Display（标题+正文），圆润衬线风格
- 布局：左侧固定导航栏 + 中央主内容区
- 图标：Lucide图标库
- 特效：磨砂玻璃毛玻璃（backdrop-filter: blur(12px)）、细金线边框、CSS关键帧动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局 | 导航栏 | 半透明深色毛玻璃，60px收起/180px展开，0.3s动画，5个Lucide图标 |
| 精灵交互页 | 瓶子区域 | 中央大型瓶子CSS渲染，3D旋转交互，瓶身光晕脉冲动画 |
| 精灵交互页 | 精灵动画 | 内嵌SVG精灵，CSS关键帧动画（跳跃/摇摆/蜷缩等），表情气泡弹缩 |
| 精灵交互页 | 操作面板 | 底部毛玻璃面板，6+1操作按钮，冷却进度环 |
| 收藏图鉴页 | 卡片网格 | 毛玻璃卡片，种类色块背景，悬停上浮-4px+阴影加深 |
| 收藏图鉴页 | 筛选栏 | 顶部5个种类标签+排序切换 |
| 装饰管理弹窗 | 装饰选项 | 三列装饰位展示，选中项高亮金框，过渡0.6s |
| 每日签到面板 | 日历网格 | 7×5日历格，呼吸动画，金色星标，7天进度条 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度1024px
- 导航栏始终固定左侧
- 图鉴卡片网格自适应列数
- 移动端暂不作为主要适配目标

### 4.4 3D场景指引

- 瓶子使用CSS 3D变换实现360度旋转感（perspective + rotateY）
- 瓶内精灵为内嵌SVG + CSS动画
- 光效使用径向渐变+关键帧动画模拟
- 无需WebGL/Three.js，纯CSS+SVG实现
